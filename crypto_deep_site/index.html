常見問題

關於 Crypto Deep 的常見疑問

AI 圖形分析的準確度如何？

我們的 YOLO 模型經過大量 K 線圖訓練，能夠識別常見的技術型態。但請注意，技術分析僅供參考，不構成投資建議。建議結合其他指標綜合判斷。

系統多久更新一次分析？

不同功能有不同的更新頻率：AI 圖形分析每 30 分鐘執行一次、價格警報即時監控、鯨魚追蹤即時推送、新聞每 5 分鐘檢查更新。

可以只選擇部分功能嗎？

可以。我們提供靈活的模組化選擇，您可以根據需求選擇需要的功能。聯繫我們討論最適合您的方案組合。

支援哪些交易所的數據？

目前支援 Binance、OKX、Bybit 三大交易所的數據。不同功能支援的交易所可能有所不同，詳情請聯繫我們。

如何接收分析通知？

分析結果會透過 Discord Bot 或 Telegram Bot 發送到您指定的頻道/群組。您只需要將我們的 Bot 邀請到您的伺服器即可。

需要提供交易所 API 嗎？

不需要。我們的系統使用交易所的公開 API 獲取市場數據，不需要您提供任何私人 API 密鑰，完全不會接觸您的帳戶。
