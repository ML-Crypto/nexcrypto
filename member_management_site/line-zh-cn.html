常见问题

关于 LINE OpenChat 管理系统的常见疑问

这个系统支持哪些交易所？

目前支持 OKX、Bybit、BingX、Binance 和 Bitget 五大主流交易所。系统会自动通过交易所 API 验证用户的交易量、入金额度和 KYC 等级。如有其他交易所需求，欢迎联系我们讨论定制化方案。

用户需要如何设置才能通过验证？

用户需要将 LINE 昵称设置为交易所 UID（例如：12345678 或 12345678小明），并在加入问题中回答所使用的交易所名称（OKX、Bybit、BingX、Binance 或 Bitget）。系统会自动提取这些信息进行验证。

可以管理多个 LINE OpenChat 社区吗？

可以。系统支持同时管理多个 LINE OpenChat 社区，每个社区可以有独立的审核设置和加入问题。管理后台会整合显示所有社区的成员资料。

审核不通过的用户会怎样？

审核不通过的用户会被自动拒绝加入申请，系统会记录拒绝原因（如：交易量不足、未找到 UID 等）。用户可以在满足条件后重新申请加入。

可以在后台修改社区设置吗？

可以。您可以在后台修改社区的加入方式（公开/需审核）、加入问题等设置，修改会即时同步到 LINE。

系统如何部署？

系统可以部署在您自己的电脑或服务器上运行。我们提供完整的安装指导和技术支持，确保您能顺利启用系统。
