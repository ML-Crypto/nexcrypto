FAQ

Common questions about NEX Telegram Management System

Which exchanges are supported?

Currently supports OKX, Bybit, and BingX. The system automatically verifies user trading volume, deposit amount, and KYC level via exchange APIs. For other exchanges, contact us to discuss custom solutions.

Do I need technical background to use this?

The system is designed for ease of use with an intuitive web admin interface. Basic operations don't require programming knowledge - just a web browser. We also provide complete installation and technical support.

How is the system deployed?

The system can be deployed on your own Windows computer or server. We provide complete installation guidance and technical support to ensure smooth setup.

Is technical support included?

Yes, all plans include technical support. We provide real-time support via Telegram to help resolve any issues you encounter.

Can I try before buying?

Contact us to arrange a product demo so you can understand the full features and operation before deciding to purchase.

What happens when VIP expires?

The system automatically tracks VIP expiration. Reminders are sent before expiry, and VIP permissions are automatically removed after expiration. Users can re-verify anytime to restore VIP status.

How does the referral system work?

Each member has a unique invite code. When new users register using the invite code and successfully bind their exchange, the system automatically records the referral relationship. The admin panel shows each member's invite count, downline list, and referral leaderboard.

Do users need to provide API keys?

No. Users only need to provide their exchange UID (User ID). The system uses the exchange's public API to verify VIP eligibility - no sensitive API keys required from users.
