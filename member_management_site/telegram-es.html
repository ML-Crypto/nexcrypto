Preguntas Frecuentes

Preguntas comunes sobre el Sistema de Gestión Telegram de NEX

¿Qué exchanges son compatibles?

Actualmente soporta OKX, Bybit y BingX. El sistema verifica automáticamente volumen de trading, cantidad de depósitos y nivel KYC vía APIs de exchanges. Para otros exchanges, contáctanos para discutir soluciones personalizadas.

¿Necesito conocimientos técnicos para usar esto?

El sistema está diseñado para facilidad de uso con una interfaz web intuitiva. Las operaciones básicas no requieren conocimientos de programación - solo un navegador web. También proporcionamos instalación completa y soporte técnico.

¿Cómo se despliega el sistema?

El sistema puede desplegarse en tu propia computadora Windows o servidor. Proporcionamos guía de instalación completa y soporte técnico para asegurar una configuración sin problemas.

¿Está incluido el soporte técnico?

Sí, todos los planes incluyen soporte técnico. Proporcionamos soporte en tiempo real vía Telegram para ayudar a resolver cualquier problema que encuentres.

¿Puedo probar antes de comprar?

Contáctanos para organizar una demostración del producto para que puedas entender las funciones completas y operación antes de decidir comprar.

¿Qué pasa cuando expira el VIP?

El sistema rastrea automáticamente la expiración VIP. Se envían recordatorios antes de la expiración, y los permisos VIP se eliminan automáticamente después. Los usuarios pueden re-verificar en cualquier momento para restaurar el estado VIP.

¿Cómo funciona el sistema de referidos?

Cada miembro tiene un código de invitación único. Cuando nuevos usuarios se registran usando el código y vinculan exitosamente su exchange, el sistema registra automáticamente la relación de referido. El panel muestra conteo de invitados, lista de línea descendente y tabla de líderes de referidos.

¿Los usuarios necesitan proporcionar claves API?

No. Los usuarios solo necesitan proporcionar su UID de exchange (ID de Usuario). El sistema usa la API pública del exchange para verificar elegibilidad VIP - no se requieren claves API sensibles de los usuarios.
