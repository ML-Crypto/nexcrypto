Preguntas Frecuentes

Preguntas comunes sobre el sistema de gestión LINE OpenChat

¿Qué exchanges soporta el sistema?

Actualmente soporta OKX, Bybit, BingX, Binance y Bitget, cinco exchanges principales. El sistema verifica automáticamente volumen de trading, monto de depósito y nivel KYC via API. Para otros exchanges, contáctanos para discutir soluciones personalizadas.

¿Cómo configuran los usuarios para pasar la verificación?

Los usuarios deben configurar su apodo de LINE con su UID de exchange (ej: 12345678 o 12345678Juan), y responder la pregunta de ingreso con el nombre del exchange (Bybit, OKX, BingX, Binance o Bitget). El sistema extrae automáticamente esta información para verificar.

¿Puedo gestionar múltiples comunidades LINE OpenChat?

Sí. El sistema soporta gestionar múltiples comunidades LINE OpenChat simultáneamente, cada una con configuraciones de revisión y preguntas independientes. El panel consolida datos de miembros de todas las comunidades.

¿Qué pasa con usuarios que no pasan la revisión?

Los usuarios que no pasan son rechazados automáticamente. El sistema registra la razón del rechazo (ej: volumen insuficiente, UID no encontrado, etc.). Los usuarios pueden volver a solicitar después de cumplir los requisitos.

¿Puedo modificar la configuración de comunidad desde el backend?

Sí. Puedes modificar el método de ingreso (público/requiere aprobación), preguntas de ingreso y otras configuraciones desde el backend. Los cambios se sincronizan a LINE en tiempo real.

¿Cómo se despliega el sistema?

El sistema puede desplegarse en tu propia computadora o servidor. Proporcionamos guía de instalación completa y soporte técnico para asegurar una configuración sin problemas.
