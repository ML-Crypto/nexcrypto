常見問題

關於 NEX Telegram 會員管理系統的常見疑問

這個系統支援哪些交易所？

目前支援 OKX、Bybit 和 BingX 三大主流交易所。系統會自動透過交易所 API 驗證用戶的交易量、入金額度和 KYC 等級。如有其他交易所需求，歡迎聯繫我們討論客製化方案。

需要什麼技術背景才能使用？

系統設計以易用性為優先，提供直覺化的 Web 管理介面。基本操作不需要程式背景，只需要會使用網頁瀏覽器即可。我們也提供完整的安裝部署服務與技術支援。

系統如何部署？需要自己準備伺服器嗎？

系統可以部署在您自己的 Windows 電腦或伺服器上運行。我們提供完整的安裝指導和技術支援，確保您能順利啟用系統。

購買後有技術支援嗎？

是的，所有方案都包含技術支援。我們透過 Telegram 提供即時技術支援，協助您解決使用過程中遇到的任何問題。

可以先試用再決定購買嗎？

歡迎聯繫我們安排產品演示，讓您了解系統的完整功能與操作方式，再決定是否購買。

VIP 到期後會發生什麼？

系統會自動追蹤 VIP 有效期限。到期前會發送提醒通知，到期後系統會自動移除用戶的 VIP 權限。用戶可以隨時重新驗證來恢復 VIP 資格。

推薦系統如何運作？

每位會員都有專屬的邀請碼。當新用戶使用邀請碼註冊並成功綁定交易所後，系統會自動記錄推薦關係。管理後台可以查看每位會員的邀請人數、下線名單和推薦排行榜。

用戶需要提供 API Key 嗎？

不需要。用戶只需要提供交易所的 UID（用戶識別碼），系統會透過交易所的公開 API 來驗證用戶是否符合 VIP 資格，整個過程不需要用戶提供任何敏感的 API 密鑰。
