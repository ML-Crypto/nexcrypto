Frequently Asked Questions

Common questions about NEX VIP Manager

Which exchanges does the system support?

Currently supports OKX, Bybit, and BingX, three major exchanges. The system auto-verifies user trading volume, deposit amount, and KYC level via exchange API. For other exchange needs, contact us to discuss custom solutions.

What technical background is needed to use this?

The system is designed with ease-of-use in mind, providing an intuitive web admin interface. Basic operations don't require programming knowledge - just web browser skills. We also provide complete installation and technical support.

How is the system deployed? Do I need my own server?

The system can be deployed on your own Windows computer or server. We provide complete installation guidance and technical support to ensure smooth setup.

Is technical support included after purchase?

Yes, all plans include technical support. We provide real-time support via Telegram to help resolve any issues you encounter.

Can I try before buying?

Contact us to arrange a product demo so you can understand the complete features and operation before deciding to purchase.

What happens when VIP expires?

The system auto-tracks VIP expiry dates. Reminder notifications are sent before expiry, and the system auto-removes VIP role and channel permissions after expiry. Users can re-verify anytime to restore VIP status.

How does the referral system work?

Each member has a unique referral code. When new users register with a referral code and successfully bind an exchange, the system auto-records the referral relationship. The admin panel shows each member's referral count, downline list, and referral leaderboard.

Do I have to use both Discord and Telegram?

No. Discord is the main verification platform, Telegram integration is optional. You can use Discord alone to manage members, or enable Telegram groups to expand your community reach.

Do users need to provide an API Key?

No. Users only need to provide their exchange UID (User ID). The system verifies VIP eligibility through the exchange's public API - no sensitive API keys required from users.
